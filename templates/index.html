System Architecture

Federated learning pipeline · Privacy-preserving distributed training

Flower (flwr) Framework
Open-source FL framework used to orchestrate multi-client training. Strategy: FedAvg. Each hospital runs a flwr.client.NumPyClient.
CNN Stack
Conv2D(32) → Conv2D(64) → Conv2D(128) → Dense(256) → Dense(128) → Softmax(2). Total ~4.2M parameters. Input: 64×64×3 RGB frames.
OpenCV Pipeline
Haarcascade frontal face detector locates ROIs from video frames. Each ROI is resized to 64×64 and passed to the CNN for mask/no-mask inference.
